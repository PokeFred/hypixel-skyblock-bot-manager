# Hypixel Skyblock - Bot Manager

## 1. Requirements

- Java (JDK21 or higher)

## 2. Installation

### 2.1 Installation (User)

#### Clone the repository

```shell
$ git clone https://github.com/PokeFred/hypixel-skyblock-bot-manager.git
```

#### Open the repository

```shell
$ cd hypixel-skyblock-bot-manager
```

#### Build the application

```shell
$ ./gradlew build
```

#### Run the build artifact

```shell
$ java -jar ./build/libs/hypixel-skyblock-bot-manager.jar
```

### 2.2 Installation (Developer)

#### Clone the repository

```shell
$ git clone https://github.com/PokeFred/hypixel-skyblock-bot-manager.git
$ cd hypixel-skyblock-bot-manager
$ ./gradlew init
```

#### Run the application

```shell
$ ./gradlew run
```

#### Build the application

```shell
$ ./gradlew build
$ java -jar ./build/libs/hypixel-skyblock-bot-manager.jar
```

#### Test the application

```shell
$ ./gradlew test
```
